This is a demonstration of the jhCodeMirror directive.

It can be used to automatically reflect the inner markup of a containing div into a nice markup box.

The calling card widget example above is inside a div with the directive applied as an attribute:

<div class="sample" jh-code-mirror="mirror">
    <!-- widget code goes inside here -->
</div>

The attribute value refers to the markup box element id. This allows multiple mirrored divs on the same page if necessary. The markup box this refers to is located further down the page and looks like:

<div class="mirror" id="mirror">
    <div class="title">
        Markup
    </div>
    <div class="block">
        <code class="prettyprint"></code>
    </div>
</div>

The code element has the class prettyprint as the demonstration also makes use of Google's code-prettify script to render the text with all the contextual colours.
